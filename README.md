### Instalação  :computer:<br/><br/>
**Clone o Repositório**
```bash
git clone https://github.com/pedrolucaspdev/cepv2-brasil
```

**Instale as Dependências** :fire:
```bash
yarnpkg ou npm install
```

**Agora é só executar**
```bash
yarnpkg dev ou npm run start
```
**Server Default: 127.0.0.1:3000**
